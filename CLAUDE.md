# Boxing & Kickboxing — сайт запису на тренування

Статичний сайт для запису клієнтів на персональні тренування. Власник — тренер з боксу/кікбоксингу (Львів).

## Стек
- Чистий HTML/CSS/JS (без фреймворків), ES-модулі через CDN `gstatic`.
- Дані: **Firebase Firestore**, колекція `bookings`. Проєкт: `boxing-booking-17e44`.
- Real-time оновлення слотів через `onSnapshot`.
- Хостинг: **GitHub Pages** — https://arturov333.github.io/boxing-booking/
- GitHub repo: https://github.com/Arturov333/boxing-booking (акаунт Arturov333, гілка main)
- Деплой оновлень: `git add . && git commit -m "..." && git push` у теці проєкту (gh CLI у `C:/Program Files/GitHub CLI`).
- Локальний тест: `python -m http.server 8000`.

## Файли
- `index.html` — розмітка
- `style.css` — стилі (Fight Brand)
- `script.js` — логіка + Firebase. Усі константи нагорі у блоці CONFIG.
- `README.md` — інструкція з налаштування Firebase

## Ключові налаштування (вгорі script.js)
- `ADMIN_CODE` = `13579` (код тренера для входу в кабінет)
- `LOCATIONS` — 2 зали: «Замарстинівська 55Б», «ТЦ Спартак»
- `TIME_SLOTS` — 12:00–16:00
- Дні: Пн–Сб (без неділі), до 4 тижнів наперед

## Важлива поведінка
- Кабінет тренера показує ТІЛЬКИ майбутні записи (минулі відфільтровані).
- Запис не збережеться, якщо Firestore Rules не опубліковані (мають дозволяти read + create для `bookings`).

## Дизайн
- Стиль: преміум, темний драматичний герой + графіт/білий контент, єдиний акцент — боксерський червоний (#E5392F). Шрифт Inter.
- **Темна/світла тема** — перемикач у герої (#theme-toggle), збереження в localStorage, дефолт dark. Початкова тема ставиться інлайн-скриптом у <head> (анти-флеш).
- **Рухомий фон** — анімовані radial-gradient «блоби» (body::before, @keyframes drift), різні кольори на тему.
- Герой: центрована мотиваційна цитата «Шлях здолає той, хто *йде*» (Playfair Display, італік, акцент на «йде»). Анімацію груші/рукавиць ПРИБРАНО (власник просив дорожчий, стриманіший стиль).
- Шрифти: Inter (контент) + Playfair Display (цитата/стрічка).
- **Рядок чемпіонів** (marquee): Усик/Кроуфорд/Головкін/Ломаченко/Кличко — лише ІМЕНА як натхнення.
- ⚠️ Реальні фото боксерів НЕ використовувати (копірайт + право на образ). Хоче колись — тільки ліцензія (Getty) або власні зйомки.
- Вибраний слот = заливка var(--select) з акцентною смужкою згори.
