# Boxing & Kickboxing — сайт запису на тренування

Статичний сайт для запису клієнтів на персональні тренування. Власник — тренер з боксу/кікбоксингу (Львів).

## Стек
- Чистий HTML/CSS/JS (без фреймворків), ES-модулі через CDN `gstatic`.
- Дані: **Firebase Firestore**, колекція `bookings`. Проєкт: `boxing-booking-17e44`.
- Real-time оновлення слотів через `onSnapshot`.
- Хостинг: наразі локально (`python -m http.server 8000`). План — GitHub Pages.

## Файли
- `index.html` — розмітка
- `style.css` — стилі (Fight Brand)
- `script.js` — логіка + Firebase. Усі константи нагорі у блоці CONFIG.
- `README.md` — інструкція з налаштування Firebase

## Ключові налаштування (вгорі script.js)
- `ADMIN_CODE` = `13579` (код тренера для входу в кабінет)
- `LOCATIONS` — 2 зали: «Замарстинівська 55Б», «ТЦ Спартак»
- `TIME_SLOTS` — 12:00–16:00
- Дні: Пн–Сб (без неділі), до 4 тижнів наперед

## Важлива поведінка
- Кабінет тренера показує ТІЛЬКИ майбутні записи (минулі відфільтровані).
- Запис не збережеться, якщо Firestore Rules не опубліковані (мають дозволяти read + create для `bookings`).

## Дизайн
- Стиль: **преміум-мінімал** (графіт + білий, тонкі hairline-лінії, багато повітря, шрифт Inter, заокруглені картки, м'які тіні). Обрано власником.
- Вибраний слот = графітова заливка (не червона).
