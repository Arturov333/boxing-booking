# Тижневі протоколи для клієнтів

Кожен файл = окремий тиждень окремого клієнта. Самодостатній, без бекенду.

## Як зробити новий тиждень (3 кроки)
1. Скопіюй `_SHABLON.html` і назви латиницею: `імя-номер.html` (напр. `oleg-1.html`, `artur-2.html`).
2. Відкрий копію, угорі зміни блок `const DATA = {…}`: `week`, `client`, дні/вправи/примітки. Дизайн НЕ чіпай.
3. Запуш у GitHub (`git add protokol/ ; git commit -m "новий протокол" ; git push`) — Vercel за ~20с викладе сам.

## Посилання клієнту
`https://boxing-booking.vercel.app/protokol/імя-номер.html`
(напр. `…/protokol/artur-1.html`)

## Підказки по даних
- День відпочинку: додай `rest: true` (плашка стане сірою).
- Без примітки: лиши `note: ""`.
- Можна будь-яку кількість блоків і вправ у дні.
